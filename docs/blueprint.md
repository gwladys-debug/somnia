# **App Name**: Noctura Sleep Aid

## Core Features:

- Sleep Habit Assessment: An interactive 4-step questionnaire analyzing sleep duration, stress levels, physical activity, and stimulant consumption.
- AI Sleep Coaching Tool: An AI tool that reasons over your assessment responses to generate ultra-personalized behavioral recommendations for better rest.
- Progress Tracking: Securely store your assessment results over time in a Cloud Firestore database for historical insight.
- Visual Results Dashboard: A 2x2 grid view displaying specialized recommendations for each sleep pillar in high-contrast light blue cards.
- Adaptive Progress System: Real-time navigation with a micro-progress bar to track movement through the 4-pillar questionnaire tunnel.
- Session Persistence: Integrated state management ensuring users can pick up their test where they left off.

## Style Guidelines:

- Primary color: Deep Midnight Blue (#0D47A1) for a high-contrast text and interactive focus.
- Background color: Dark Charcoal Grey (#212529) providing a low-light, eye-stretching experience suitable for insomniacs.
- Accent: Pearl Grey (#E0E0E0) for buttons and active states to maintain visibility without causing eye fatigue.
- Headline and body font: 'Poppins' for its contemporary geometric feel and clean readability on small mobile screens.
- A strictly mobile-first architecture featuring centered containers and high-density 16px border-radius cards for a tactile, app-like feel.
- Smooth transitions for elevation changes on buttons and card entries to enhance the feeling of calm and quality.
- Minimalist geometric icons represent each pillar (Sleep, Relaxation, Sport, Drinks) to provide visual shortcuts for content scanning.
